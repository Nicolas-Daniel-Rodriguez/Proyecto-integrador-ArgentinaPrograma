<p id="description">Web application for portfolio. In it you can see personal academic data, work skills and projects along with your own aspirations. While logged in, you can create new fields, edit and delete them. Angular was pulled for the Front-End and hosted on Firebase. For the Database, MYSQL is used and it is hosted on Clever Cloud. As for the Back-End, SpringBoot is used and it is hosted on Heroku.</p>
